Arquitectura del Proyecto

Java Spring Boot — Hexagonal + Vertical Slicing

Estructura de paquetes
com.medicitas/
├── shared/                        ← Infraestructura compartida
│   ├── config/
│   │   ├── SecurityConfig.java
│   │   └── JwtConfig.java
│   └── exception/
│       └── GlobalExceptionHandler.java
│
├── consultorios/                  ← Vertical Slice: Consultorios
│   ├── domain/
│   │   ├── Consultorio.java       (Entidad)
│   │   └── ConsultorioPort.java   (Puerto salida)
│   ├── application/
│   │   ├── ConsultorioService.java
│   │   └── dto/
│   │       ├── ConsultorioRequestDto.java
│   │       └── ConsultorioResponseDto.java
│   └── infrastructure/
│       ├── ConsultorioController.java  (Puerto entrada HTTP)
│       ├── ConsultorioRepository.java  (Oracle JPA)
│       └── ConsultorioMapper.java
│
├── medicos/                       ← Vertical Slice: Médicos
│   ├── domain/
│   │   ├── Medico.java
│   │   └── MedicoPort.java
│   ├── application/
│   │   ├── MedicoService.java
│   │   └── dto/
│   └── infrastructure/
│       ├── MedicoController.java
│       └── MedicoRepository.java
│
└── auth/                          ← Vertical Slice: Auth/Roles
    ├── domain/
    │   └── RolPermiso.java
    ├── application/
    │   └── AuthService.java
    └── infrastructure/
        ├── AuthController.java
        └── JwtFilter.java
Puertos y Adaptadores
ENTRADA (Driving)
REST Controllers (HTTP)
Request DTOs + Validaciones

SALIDA (Driven)

Port interfaces → impl

DOMINIO (Núcleo)
Entidades puras
Reglas de negocio
Sin dependencias externas
